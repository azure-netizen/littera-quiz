# Social Features Requirements

## Purpose

Support score sharing, referrals, and achievement visibility to increase participation and engagement.

## Users

- Student
- Teacher

## Functional Requirements

- Users shall be able to share scores on WhatsApp.
- Users shall be able to share eligible achievements.
- The system shall support referral links or codes.
- The system shall track referral source.
- Achievement badges shall be visible on user profiles.
- Shared content shall include relevant score, rank, badge, or certificate details.

## Acceptance Criteria

- A student can share a quiz score using WhatsApp.
- A user can share an earned certificate or badge.
- Referral links are tracked to the referring user.
- Shared content does not expose private user data beyond approved fields.

## Dependencies

- Mobile app native sharing
- Rewards system
- User profile
